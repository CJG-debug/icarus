## 安装

```
sudo apt install vsftpd
```

## 匿名登录

**/etc/vsftp.cnf**

```
anonymous_enable=YES
```

<!-- more -->

## 默认访问地址

```
cat /etc/passwd|grep ftp
```
**/srv/ftp**

## 重启

```
systemctl restart vsftpd.service
```
