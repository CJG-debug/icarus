## 缩进

**命令行模式**

```bash
:2,4>
```

**小提示：>>相当于2次缩进**

**效果**

```bash
fun_print(){
    echo "我在干什么"
    echo "问你呢"
    echo "我怎么知道"    
}
```

## 回缩

**命令行模式**

```bash
:2,4<
```

**效果**

```bash
fun_print(){
echo "我在干什么"
echo "问你呢"
echo "我怎么知道"    
}
```
