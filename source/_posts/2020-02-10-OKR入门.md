## 概念

```
是一套明确和跟踪目标及其完成情况的管理工具和方法。
```

## 周期

```
OKR在执行过程主要以季度为周期来进行管理。
```

{% asset_img 1.webp %}

<!-- more -->

```
在季度初期开「OKR会议」，「确定OKR」，「公示OKR」，中间会进行跟踪状态目标，季度末会进行反思复盘，进行打分等等操作
```
## PDCA

```
计划，执行，检查，处理
```

## 适用

```
OKR更适合那种大方向性，那种不易于考核的团队，比如研发，业务支撑团队
```
